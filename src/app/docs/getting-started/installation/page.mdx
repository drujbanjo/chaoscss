# How to install Chaoscss

#### СhaosСss is a simple, easily modified CSS library that will take you from prototype to finished project

<br />

| variants        | practice                                                      |
| :-------------- | :------------------------------------------------------------ |
| CDN             | https://cdn.jsdelivr.net/npm/chaoscss@1.1.2/css/chaos.min.css |
| Package Manager | npm install chaoscss@1.1.2                                    |

<br />
<br />

### Quick Start with CDN

To enable ready-to-use CSS files using a CDN, you first need to complete a few steps.

1.  ###### Create your **index.html** in your project root and add <code> \<meta name="viewport" \></code>.

    <br />

    ```xml
    <html lang="en">
    	<head>
    		<meta charset="utf-8" />
    		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
    		<title>Your document name</title>
    	</head>
    	<body>
    		<!-- ... -->
    	</body>
    </html>
    ```

    <br />

    <br />

1.  ###### Put <code>\<link\></code> in <code>\<head\></code> on top of the rest of the styles.

    <br />

    ```xml
    <html lang="en">
    	<head>
    		<meta charset="utf-8" />
    		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
    		<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/chaoscss@1.1.2/css/chaos.min.css"/>
    		<!-- your styles -->
    		<title>Your document name</title>
    	</head>
    	<body>
    		<!-- ... -->
    	</body>
    </html>
    ```

<br />

By taking these simple steps, you can quickly start using Chaoscss.

<br />

### Start with Package Manager

To use chaoscss library with package manager you need to follow few steps.

1. ###### Install ChaosCss through the package manager.

   <br />

   ```bash
   npm install chaoscss
   ```

   <br />

1. ###### To use an installed library, you can use <code>\@import</code> in the <code>CSS</code> file and <code>\@use</code> in <code>SCSS</code>.

   {' '}

   <br />

   ```css
   @import "chaoscss";
   ```

   <br />

   ```scss
   @use "chaoscss";
   ```
